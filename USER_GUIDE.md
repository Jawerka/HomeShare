# HomeShare — краткое руководство

## Windows

1. Установите HomeShare, разрешите firewall.
2. Откройте приложение → **Устройства** → покажите PIN другому устройству или введите его PIN + IP.
3. В настройках выберите папку для входящих.
4. ПКМ по файлу в Проводнике → **HomeShare** → выберите адресата.
5. Прогресс — в окне «Передачи» и в подсказке трея (`42%`).

## Android

1. Установите APK, разрешите уведомления и сеть.
2. Привяжите устройство по PIN.
3. В любом приложении: **Поделиться** → HomeShare → выберите адресата.
4. Прогресс в шторке уведомлений и в приложении.

## Linux

1. Установите hub (`install-hub-ubuntu.sh`).
2. Откройте `http://<ip>:8787` — QR/PIN, список устройств, лог.
3. Папка inbox задаётся в `/etc/homeshare/config.json`.
4. Отправка: `homeshare-hub send --to <имя> /path/file`.
